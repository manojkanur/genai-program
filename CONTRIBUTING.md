# Contributing

Thanks for helping improve the GenAI Program!

## Ways to contribute
- Fix typos or broken links
- Improve explanations or add diagrams
- Add new exercises or example projects
- Share your buildathon / final project as a reference

## Workflow
1. Fork the repo.
2. Create a branch: `feat/<short-description>` or `fix/<short-description>`.
3. Commit with a clear message.
4. Open a PR against `main` describing **what** changed and **why**.

## Style
- Keep examples small and runnable.
- Pin dependency versions where possible.
- Prefer free / open-source tools as the default path.
