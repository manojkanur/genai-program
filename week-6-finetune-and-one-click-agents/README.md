# Week 6 — Fine-tune & One-Click AI Agents

Goal: own the model layer, and package agents so anyone can run them.

## Topics
1. **Fine-tuning basics** — when to fine-tune vs RAG vs prompt
2. **LoRA / QLoRA** — efficient fine-tuning on consumer GPUs / Colab
3. **Datasets** — collection, cleaning, format (SFT, DPO)
4. **One-click AI agents** — package as a single command / template / app

## Deliverable
A fine-tuned small model + a one-click installable agent that uses it.
