# Buildathon — Offline Project

A focused, in-person (or scheduled-online) build sprint. Apply Weeks 1–3 to ship something real.

## Format
- Team size: 1–3
- Duration: 1 day (suggested)
- Theme: Solve a real workflow problem with GenAI

## Folders
- `problem-statements/` — curated prompts to pick from
- `starter-template/` — minimal FastAPI + Streamlit + LangChain scaffold
- `submissions/` — each team creates a subfolder with their project + README

## Judging rubric (suggested)
| Criteria | Weight |
|---|---|
| Problem fit | 25% |
| Technical execution | 25% |
| UX / Demo polish | 20% |
| Evals & reliability | 15% |
| Creativity | 15% |
