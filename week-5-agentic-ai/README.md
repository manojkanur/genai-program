# Week 5 — AI That Thinks On Its Own

**Goal:** Build AI agents that plan, decide, and finish complex jobs by themselves.

## What you'll learn
- **Agent designs** — different ways agents think (ReAct, Plan-and-Execute, etc.)
- **Multiple agents working together** — like a team
- **Memory** — agents that remember across days, not just one chat
- **Real projects** — build something useful

## Your project this week
Build an AI agent that:
- Has its own goal
- Uses at least 3 tools
- Finishes the task without you guiding each step

Example: a research agent that picks a topic, searches the web, reads pages, and writes a report.

## Folders
- `01-agent-architectures` — types of agents
- `02-multi-agent-systems` — teams of agents
- `03-memory-and-state` — long-term memory
- `04-projects` — build something real
