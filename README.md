# 6-Week GenAI Program

A community-driven, hands-on program to take you from Python basics to shipping your own MicroSaaS powered by GenAI.

> Maintained for the community. Contributions, issues, and discussions welcome.

## Week 1 — Python & GenAI Foundation
- Python essentials
- Platform setup (env, IDE, accounts)
- FastAPI (your first API)
- Streamlit (your first UI)
- RPA with `pyautogui` and `playwright`
- Open-source vs closed-source LLMs
- Running models locally with Ollama

## Week 2 — Prompt & LangChain
- Prompt engineering patterns
- LangChain core concepts
- Chains, memory, output parsers
- Tools and structured outputs

## Week 3 — RAG, Guardrails, Evals
- RAG pipelines end-to-end
- Vector databases
- Guardrails (input/output safety)
- Evaluations (offline + online)
- Take your project **live**

## Buildathon (Offline Project)
- Problem statements
- Starter template
- Judging rubric
- Submission folder

## Week 4 — AI Automation & AI Agents
- Automation patterns
- Agents 101
- Tool calling
- Multi-step workflows

## Week 5 — Agentic AI
- Agent architectures (ReAct, Plan-and-Execute, etc.)
- Multi-agent systems
- Memory and state
- Capstone-style projects

## Week 6 — Fine-tune & One-Click AI Agents
- Fine-tuning basics
- LoRA / QLoRA
- Dataset preparation
- Packaging "one-click" agents

## Final — MicroSaaS Template
- Frontend / Backend / Infra scaffolding
- Auth, billing, deployment notes
- Use this to ship your own product

---

## How to use this repo
1. Fork or clone the repo.
2. Work through each week in order — every folder has its own `README.md`.
3. Commit your solutions in your own fork and share them in the community.
4. Open issues / PRs to improve the material.

## Tech Stack (default choices)
- Python 3.11+
- FastAPI, Streamlit
- LangChain, Ollama
- OpenAI / Anthropic / open-source LLMs
- Vector DB (Chroma / Qdrant)
- Vercel for deployment of the MicroSaaS frontend
