# Week 4 — Make AI Do Tasks For You

**Goal:** Move from "AI that talks" to "AI that does work."

## What you'll learn
- **Automation** — connect AI to your email, calendar, sheets, etc.
- **AI Agents** — AI that can think in steps and use tools
- **Tool calling** — let AI use real tools (send email, search web, edit a file)
- **Workflows** — multi-step tasks AI completes on its own

## Your project this week
Build something that does a task end-to-end without you clicking.
Examples:
- Summarize today's emails into Slack
- Read a PDF and create a to-do list
- Find new leads on LinkedIn and save them to a sheet

## Folders
- `01-ai-automation` — automation basics
- `02-ai-agents-intro` — what an agent is
- `03-tool-calling` — give AI real tools
- `04-multi-step-workflows` — putting it all together
